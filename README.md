# Skyglass Isles

A small 3D isometric action RPG prototype built with Three.js. Move the player around the floating island, avoid stone blocks, collect every crystal, and slash toward the cursor.

## Run

```powershell
npm install
npm run build
npm run dev
```

Then open:

```text
http://localhost:5173
```

## Controls

- Hold arrow keys or WASD: move smoothly
- Left click: attack
- Reset button: restart the run

## Development

- Source: `src/game.ts`
- Browser build: `dist/game.js`
- Type-check only: `npm run check`
